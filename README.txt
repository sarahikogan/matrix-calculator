Matrix Calculator

SUMMARY: 
This program accepts two matrices and then performs an operation on them, 
either addition, subtraction, or multiplication. It also catches errors.

TO RUN: 
1. find the file in the terminal
2. compile the file using gcc matrix_calc.c 
3. run the file using ./a.out
4. enter the size of matrix A, using the format "X Y" (x being the rows, y 
being the columns)
5. Enter the values for matrix A 
6. repeat the process for matrix B 
7. Choose an operation 
The program will print the matrix for the operation, properly spaced.
